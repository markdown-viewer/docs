# Edge Extension

Markdown Viewer is available from the Microsoft Edge Add-ons store for users who want the native Edge installation and update path.

## Overview

| Feature | Status |
|---------|--------|
| Markdown rendering | ✅ |
| Word export | ✅ |
| All diagram types | ✅ |
| LaTeX formulas | ✅ |
| 29 themes | ✅ |
| Smart caching | ✅ |
| Offline mode | ✅ |

**Version:** 5.0.0  
**Store:** Microsoft Edge Add-ons

---

## Installation

### From Edge Add-ons (Recommended)

1. Visit [Edge Add-ons - docu.md Markdown Viewer](https://microsoftedge.microsoft.com/addons/detail/documd-markdown-viewer/iphmkjlbnogmhofmmcahdhodiilokfca)
2. Click **Get**
3. Confirm **Add extension** in Edge
4. Pin the extension from the toolbar if you want quick access

### Alternative: Chrome Web Store Compatibility

Most Chromium-based features are identical. If your environment allows Chrome Web Store installs in Edge, the Chrome listing also works, but the Edge Add-ons page is the official Microsoft Edge route.

## Notes

- Uses the same rendering and export engine as the Chrome extension
- Updates are managed through Microsoft Edge Add-ons
- For local files, enable **Allow access to file URLs** in the extension details page
